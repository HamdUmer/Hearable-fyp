# HearAble
  .
A Flutter project for the HearAble application.

## Prerequisites

Before you begin, ensure you have met the following requirements:

*   **Flutter SDK**: [Install Flutter](https://docs.flutter.dev/get-started/install) for your operating system.
*   **Editor**: VS Code, Android Studio, or IntelliJ IDEA with the Flutter and Dart plugins installed.
*   **Device**: An Android Emulator, iOS Simulator, or a physical device connected via USB.

## Setup Instructions

1.  **Navigate to the project directory**:
    ```bash
    cd HEARABLE
    ```

2.  **Install Dependencies**:
    Run the following command in the root directory to fetch all necessary packages:
    ```bash
    flutter pub get
    ```

## Running the Application

To run the app on your connected device or emulator:

```bash
flutter run
```

### Platform Specifics

*   **Android**: Ensure you have an emulator running or a device connected with USB debugging enabled.
*   **iOS**: Open the simulator or connect an iOS device (requires macOS).
*   **Web**: Run `flutter run -d chrome` to view in the browser.
*   **Windows/Linux/macOS**: Run `flutter run -d windows` (or `linux`/`macos`) for desktop support.

## Troubleshooting

If you encounter issues, try running:

```bash
flutter doctor
```

This command checks your environment and displays a report of the status of your Flutter installation.
